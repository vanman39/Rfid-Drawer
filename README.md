# Rfid-Drawer
An rfid locked drawer that opens with the correct rfid card 
BOM
-1 esp32 wroom
-jumper wires
-1 rfid sensor
-1 keyfob or keycard
-3d printed drawer (see stls)
-1 5g servo
-1 1000 uf capacitor 10v min 16 to 25v is great
-1 430 pin breadboard with both rails
NOTE: the servo is on 5v and the rfid is on 3.3v and the esp32 is pushed into a 430 pinbreadboard 
Libraries for codebase: ESP32Servo and MFRC522
